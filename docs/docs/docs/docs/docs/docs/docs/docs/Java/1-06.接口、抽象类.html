接口、抽象类

接口和抽象类有什么区别？

共同点

  • 是上层的抽象层。
  • 都不能被实例化。
  • 都能包含抽象的方法，这些抽象的方法用于描述类具备的功能，但是不能提供具体的实现。

区别

  • 在抽象类中可以写非抽象的方法，这样可以提高代码的复用性，这是抽象类的优势，接口中只能有抽象的方法。
  • 一个类只能继承一个直接父类，这个父类可以是具体的类也可是抽象类，但是一个类可以实现多个接口。
